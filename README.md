# Physio Algo Project

## Overview
This project applies basic algorithms to physiotherapy data such as pain scores, recovery timelines, and exercise progression.

## Algorithms Used
- Fibonacci (Recursive & DP)
- Sorting (Bubble, Selection, Insertion)
- Binary Search

## Applications
- Exercise planning
- Patient data sorting
- Fast record searching

## Tools
Python, NumPy, Matplotlib
